# Leia+

Leia+ é um sistema simples e direto que permite gerenciar o acervo de livros de uma biblioteca comunitária, registrar empréstimos/devoluções e permitir que usuários consultem os livros disponíveis

## 🛠️ Ferramentas

* HTML
* CSS
* JavaScript
* Visual Studio Code
* Git
